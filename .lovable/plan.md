
# خطة بناء برنامج تصميم المطابخ الاحترافي

نظراً لضخامة المشروع، سيتم بناء **هيكل متكامل قابل للاستخدام** في هذه المرحلة، مع إمكانية تعميق كل قسم لاحقاً.

## ملاحظة هامة حول الواتساب المجاني
لا يوجد API واتساب رسمي مجاني وتلقائي. سأستخدم الحل العملي التالي:
- **التسجيل برقم الهاتف + كلمة مرور** مع تفعيل تجريبي تلقائي 7 أيام (بدون موافقة أدمن)
- زر "تواصل عبر واتساب" لطلب التفعيل الدائم بعد انتهاء التجربة (رابط wa.me يفتح محادثة الأدمن تلقائياً)
- هذا مجاني 100% وبدون أي API مدفوع

## التصميم البصري
مزج بين الطابعين: **داكن صناعي مع لمسات ذهبية وبرتقالية دافئة** — يعطي إحساس احترافي وفاخر في نفس الوقت. خط عربي عصري (Cairo/Tajawal).

## الأقسام المنفذة

### 1. البنية التحتية (Lovable Cloud)
- قاعدة بيانات: `profiles`, `user_roles`, `subscriptions`, `designs`, `board_inventory`, `rod_inventory`, `cutting_jobs`
- RLS مع جدول أدوار منفصل (admin/user)
- Trigger لإنشاء profile + اشتراك تجريبي 7 أيام تلقائياً عند التسجيل

### 2. المصادقة
- صفحة تسجيل/دخول برقم الهاتف + كلمة المرور (عبر Supabase Auth بصيغة `phone@kitchen.app`)
- حماية المسارات: `_authenticated` layout يتحقق من صلاحية الاشتراك
- صفحة "انتهت التجربة" بزر واتساب للتفعيل

### 3. محرر التصميم (2D + 3D)
- **2D Canvas** (Konva.js): رسم الغرفة، حوائط، إدراج الوحدات كبلوكات، سحب وإفلات، تحريك على الحوائط
- **3D Viewer** (Three.js + React Three Fiber): عرض ثلاثي الأبعاد مباشر للتصميم
- مكتبة بلوكات جاهزة: قاعدة سفلية، وحدة علوية، حوض، فرن، ثلاجة، طاولة جزيرة
- إدخال المقاسات بالسم/المتر مع تبديل الوحدة
- حفظ التصميمات في قاعدة البيانات

### 4. تقطيع الألواح (Boards Cutting)
- إدارة مخزون ألواح بمقاسات مختلفة (عرض × طول × سُمك × كمية)
- إدخال قائمة قطع مطلوبة
- خوارزمية تقطيع أمثل (Guillotine/Bin Packing) + حساب الهدر
- عرض مرئي للوحات بعد التقطيع + تصدير PDF

### 5. تقطيع الأعواد (Rods Cutting)
- مخزون أعواد بأطوال مختلفة
- خوارزمية 1D Bin Packing
- تقرير الاستخدام والهدر

### 6. لوحة الأدمن
- إحصائيات: عدد المستخدمين، الاشتراكات النشطة، التصميمات
- إدارة المستخدمين: تفعيل/تجميد، تمديد الاشتراك، تعيين أدوار
- إدارة الإعلانات: تفعيل/تعطيل AdSense + إدخال الكود

### 7. الإعلانات
- مكون `<AdSlot>` يقرأ كود AdSense من إعدادات الأدمن
- يظهر للمستخدمين العاديين فقط (يختفي للأدمن)
